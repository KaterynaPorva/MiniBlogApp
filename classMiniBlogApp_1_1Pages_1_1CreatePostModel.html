IActionResult MiniBlogApp.Pages.CreatePostModel.OnPost ( )

Handles POST requests to create a new post.

Validates that Title and PostContent are not empty. If validation fails, adds an error message and returns the same page. If valid, saves the post via BlogStorage and redirects to the MyPosts page.

Returns
IActionResult Redirects to MyPosts on success, or returns the same page on validation failure.
Exceptions
InvalidOperationExceptionIf the user is not logged in (handled via redirect).
