void MiniBlogApp.Pages.PostAnalyticsModel.OnGet ( )

Handles GET request for the post analytics page.

Loads all posts from BlogStorage, analyzes each post using PostAnalyzer, and generates summary statistics using PostStatsUtils. Populates AnalyzedPosts and Summary properties for display on the page.
