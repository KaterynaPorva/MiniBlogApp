void LoggerServiceAdvancedTests.AddLog_ShouldStoreMessagesForDifferentActions ( )

Verifies that LoggerService stores messages for different types of user actions.

Adds logs for a post creation, a like, and a comment. Retrieves all log messages and asserts that:

  1. The total number of log messages matches the expected count.
  2. Each type of action has an appropriate log message.
    Returns
    void
    Exceptions
    Xunit.Sdk.XunitExceptionIf any assertion fails.
